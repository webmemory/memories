css 소개2-세부적인 식별자 class, id


문자의 크기 font-size 배치 text-align:center,left,right,justify 색깔 color

식별자는 태그 클래스 아이디가 있다.
*css에서 html 태그의 속성인 클래스는 . 아이디는 #으로 표시한다.
*동일한 태그에서 아이디는 한번만 사용되며 클래스는 제한없이 사용된다.
*클래스/id 이름만 같은면 될 경우 태그p를 표시하지 않고, .a, .b, #c,...로 하면 된다
*식별자가 동일한 태그의 속성일 때 id가 class보다 우선 적용되며
*식별자가 동일할 경우 나중 나온 것이 먼저 나온 것에 우선적용된다.

예)

a.문자의 크기 색깔 배치

b.문자의 크기 색깔 배치

c.문자의 크기 색깔 배치

d.문자의 크기 색깔 배치

e.문자의 크기 색깔 배치

코드 <style> p.a{font-size:10px;text-align:left;color:red} p.b{font-size:15px;text-align:left;color:grey} p.c{font-size:20px;text-align:center;color:blue} p.d{font-size:25px;text-align:left;color:pink} p.e{font-size:30px;text-align:left;color:purple} #kk{font-size:30px;text-align:left;color:black} </style> <p class="a">a.문자의 크기 색깔 배치</p> <p class="b">b.문자의 크기 색깔 배치</p> <p class="c">c.문자의 크기 색깔 배치</p> <p class="d">d.문자의 크기 색깔 배치</p> <p id="kk" class="e">e.문자의 크기 색깔 배치</p>
