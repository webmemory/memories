#인덴트할 때 커서를 라인 앞에 두거나, 일정영역을 선탁한 후
오른쪽으로 ctrl-]
외쪽으로 ctrl-[

#창나누기할 때
view>pane>...
창나누기를 하면 기존화일이 그대로 복제되며 한 쪽을 바꾸면 다른쪽도 바뀐다
창나누기 후 새로 만들어진 창을 빈창으로 하려면 new file을 선택하면 된다

#여러줄 동시선택할 때
에디터의 번호를 따라 ctrl 혹은 shift키를 누르고 마우스로 드래그한다

#앞뒤 따옴표 쉽게 붙이기
shift키를 누른채 해당 단어 화살표키이동하여 선택한 후 따옴표를 누른다

#태그편집 커서이동 쉽게하기
ctrl + alt키를 누르고 화살표로 이동한다

#여러개의 커서를 활성화하여 동시에 작업하기
ctrl키를 누를 상태에서 원하는 위치에 마우스를 클릭하면 커서가 클릭 되는 위치마다 커서가 생성된다.
그다름 ctrl키를 누름 해제한 후, 일정작업을 하면 모든 커서 부위에서 동시에 그 작업이 진행된다.
작업을 마친 후에는 esc키를 눌러서 커서를 해제한다.

#주석처리하기
주석처리하고 싶은 부분을 선택한 다음 ctrl + / 를 누르면 해당 부분이 주석으로처리/해제된다. 선택하지 않은 상태에서 누르면 커서가 있는 줄 전체를 주석으로 처리/해제한다.

Ctrl + F
찾기/바꾸기

Ctrl + Shift + F
프로젝트 전체에서 찾기, 바꾸기를 한꺼번에 할 수 있다

Ctrl + E
선택 영역을 찾기/바꾸기

Ctrl + G
라인 번호를 입력할 수 있는 창이 나오며 거기에 번호를 컨트롤 키를 누르고 커서를 아래로 이동해도 라인선택이 된다. 블럭 매칭

Ctrl + J
라인 조인: 아래쪽 라인이 올라와 현재 라인과 결합한다.

Ctrl + L
라인 선택: 커서가 아래쪽 라인으로 이동하며 줄 단위로 아래줄이 계속해서 선택된다. 컨트롤 키를 누르고 커서를 아래로 이동해도 라인선택이 된다.

Ctrl + D
현재 단어 선택: 현재 커서가 놓인 단어를 선택하며 이후 전체 범위에서 같은 단어를 계속 선택

Ctrl + Backspace, Delete
단어 별 삭제

Ctrl + Shift + K
현재 라인 삭제

Ctrl + Shift + D
현재 라인 다음 라인으로 복사, 모든 라인이 한 줄 아래로 밀린다

Ctrl + [, ]
들여쓰기

Ctrl + Alt + [, ]
코드 폴딩 토글: 현재 커서가 위치한 태그 전체를 폴딩하여 안 보이게 하거나 보이게 함

Ctrl + Shift + Alt + [, ]
전체 코드 폴딩 토글

Ctrl + ←, →
커서가 단어 별 이동

Ctrl + ↑, ↓
현재 현재 커서가 있는 라인 자체를 위, 아래로 이동

Ctrl + Alt + ↑, ↓
다중 커서 삽입: 위, 아래로, 왼쪽에서 현재 커서가 있는 위치까지의 거리와 동일한 위치로
커서를 표시한다. ()영어알파벳 1개와 한글 한 글자를 동일하게 인식하는 것에 주의)

Ctrl + Enter
현재 라인 아래로 개행: 커서가 있는 라인 아래쪽을 빈 칸으로 벌리고 그리로 커서 이동

Ctrl + Shift + Enter
현재 라인 위로 개행

Ctrl + space
코드 힌트 보기

Shift + 방향키
텍스트 선택: 좌우 화살표를 클릭하면 좌우로 한글자씩 선택, 상하로 클릭하면 한 줄씩 선택

Shift + Ctrl + ←, →
단어별 텍스트 선택

참고
Atom에디터는 컴파일(실행)기능이 따로 없기 때문에 번거로움이 있다.
따라서 이런 번거로움을 해결해줄 script라는 패키지를 이용하여 해당 기능을 추가할 수 있다.
